# 如何使用?


将上面的菜单价格手动输入到 [data] 例如: [16.2,17,17.7,18.5,18.7,26.7,32.7] 

[data]内的数字会自动重复使用以尽量达到最佳满减组合

在target = 85 输入您期望凑到的总满减金额，本次示例为85-45的满减 故target值使用85

在任意一个python3 终端运行即可得到最佳的10种组合
